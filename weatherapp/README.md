## 📁 Day-4 – SkyPulse Weather App

A live weather dashboard that fetches real-time data from the OpenWeatherMap API.

**Features:**
- 🔍 Search any city in the world
- 🌡️ Current temperature, feels-like, min/max
- 💧 Humidity, wind speed & direction, pressure, visibility, cloud cover
- 🌅 Sunrise & sunset with animated sun arc
- 🕐 Simulated 24-hour hourly forecast
- 💨 Weather Condition Index
- 🌐 Atmosphere details (dew point, sea level, ground level, wind gust)
- ⏱️ Live clock

**Tech Stack:** HTML · CSS · Vanilla JavaScript · OpenWeatherMap API

### Setup

1. Clone the repo
2. Copy `Day-4/config.example.js` → `Day-4/config.js`
3. Add your free API key from [openweathermap.org](https://openweathermap.org/api)
4. Open `Day-4/fetch.html` in your browser

> ⚠️ Never commit `config.js` — it's in `.gitignore` to keep your API key safe.
